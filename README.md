# The Secret Next Door

A hybrid exhibition experience for the **Pinacoteca Nazionale di Bologna**.
Course project for *Digital Heritage and Multimedia*.

**Group GGU — Global Girlies United:**
Polyxeni Chasanai · Kübra Topçuoğlu · Yixuan Qu · Saya Saito

## The website

The main page (`index.html`) has four parts:

1. **Home** — the idea and the five star paintings.
2. **Presentation** — the project slides.
3. **Twine** — the interactive story.
4. **Figma** — the UI design.

The **Design Brief** is a separate page (`design-brief.html`), opened from the navbar.
It follows the course paper template; the content fields are empty for now and will be
filled in later by the team.

The site is published with **GitHub Pages** at:
`https://yixuanqu.github.io/Digital-heritage-and-multimedia/`

## How to add the missing links

Three places use a placeholder. Open `index.html` and edit the `src` of the iframe:

- **Presentation** — paste your Google Slides / Canva *embed* link.
- **Figma** — paste your Figma *embed* link
  (`https://www.figma.com/embed?embed_host=share&url=YOUR_LINK`).
- **Twine** — in the Twine editor choose *Publish to File*, save the file as
  `story.html` in this folder. The story then plays inside the page.
